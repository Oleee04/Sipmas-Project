# 🔐 Penetration Testing

Repository ini berisi project **Penetration Testing** untuk pembelajaran dan simulasi pengujian keamanan aplikasi web pada lingkungan lokal (localhost). Project ini mengimplementasikan berbagai skenario kerentanan berdasarkan **OWASP Web Security Testing** sebagai media praktik identifikasi, eksploitasi, analisis risiko, dan mitigasi keamanan.

> **Hanya untuk tujuan edukasi, penelitian, dan pengujian keamanan yang sah.**

## 🎯 Fitur

- Simulasi **18 jenis kerentanan** aplikasi web.
- Mengikuti metodologi **OWASP Web Security Testing**.
- Dokumentasi pengujian, analisis risiko, dan rekomendasi mitigasi.
- Cocok sebagai media belajar penetration testing dasar.

## 🛠 Teknologi

- PHP Native
- MySQL
- HTML, CSS, JavaScript
- Bootstrap
- Laragon

## 🔍 Modul Pengujian

- Brute Force
- Command Injection
- CSRF
- File Inclusion
- File Upload
- Insecure CAPTCHA
- SQL Injection & Blind SQL Injection
- Weak Session IDs
- DOM, Reflected & Stored XSS
- CSP Bypass
- JavaScript Attacks
- Authorization Bypass
- Open Redirect
- Cryptography
- API Security Testing

## 🚀 Menjalankan Project

```bash
git clone https://github.com/username/penetration-testing.git
```

1. Pindahkan project ke `htdocs/`
2. Import database
3. Konfigurasi `config/database.php`
4. Jalankan Apache & MySQL
5. Akses:

```
http://localhost/penetration-testing
```
